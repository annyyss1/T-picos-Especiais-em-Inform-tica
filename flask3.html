O QUE É FRAMEWORK?

Em primeiro lugar frameworks são estruturas que servem de base para a construção de aplicações. Os benefícios de usá-los são inúmeros, desde uma maior otimização de tempo até o impacto positivo em todos os setores de uma companhia, uma vez que o tempo dos profissionais gastos em códigos reescritos pode ser voltado à inovação.

FRAMEWORKS: FLASK

Micro framework que utiliza a linguagem Python para criar aplicativos Web. Foi Lançado em 2010 e desenvolvido por Armin Ronacher, é destinado a pequenas aplicações com requisitos mais simples, como por exemplo, a criação de um site básico.

CARACTERÍSTICAS

  • Simplicidade;
  • Rapidez;
  • Soluções para projetos pequenos;
  • Aplicações robustas.
